# food_deliveryapp
 1. Problem Statement

 “Design and implement a food delivery system using object-oriented programming that manages restaurants, menu items, cart operations, order processing, delivery assignment, and user ratings.”

 2. Approach / Logic Used

“The system is designed using OOP concepts where each real-world entity is represented as a class. The flow includes adding items to a cart, applying promo codes, placing an order, assigning a delivery partner, and submitting ratings. Methods handle operations like price calculation, availability checking, and validation.”

3. Steps to Execute the Code

“The execution follows these steps:”

Create a restaurant object
Create menu items
Add items to the restaurant menu
Create a cart for the customer
Add items to the cart
Apply promo code (optional)
Calculate total using checkout()
Create an order with cart items
Place the order if restaurant is open
Calculate ETA based on preparation time
Assign a delivery partner
Update delivery status
Mark order as delivered
Submit rating and review
